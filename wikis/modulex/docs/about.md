# About FreeSTAR ModuleX

FreeSTAR ModuleX is a modular component system designed for building flexible amateur radio solutions.

## Mission

The mission of FreeSTAR ModuleX is to provide licensed amateur radio operators with modular, extensible components for custom communication systems.

## Features

- Modular architecture
- Easy integration and extension
- Compatible with FreeSTAR ecosystem
