# About FreeSTAR Multi-Mode

FreeSTAR Multi-Mode is a versatile amateur radio communication platform designed to support multiple modes of operation.

## Mission

The mission of FreeSTAR Multi-Mode is to provide licensed amateur radio operators with flexible and powerful tools for multi-mode communication.

## Features

- Support for various communication modes
- Easy integration with existing systems
- Licensed amateur radio operator focused
